# OpenGL-Basic-applications-2

Transforms:
` - Nothing
1 - Translate
2 - Scale
3 - Rotate 

Draws:
4 - nothing
5 - cube
6 – pyramid
7 – hexagonal prism

Camera moves:
Up Arrow Key    - forward
Down Arrow Key  - backwards
Left Arrow Key  - to the left
Right Arrow Key - to the right
W Key           - up
S Key           - down
Mouse           - look around
